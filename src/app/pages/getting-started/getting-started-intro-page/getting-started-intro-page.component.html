Getting Started

Srcry is a css utility that introduces a means of behavioral scaling to responsive design. As the Internet evolved, we had to grow with it by focusing less on the exact sizes of our elements and more on the way it should transition from one size and sometimes shape to another across various screens. It’s safe to say responsive design is organically becoming more and more of a type of “animation programming” in its’ own unique right with all the varying factors we have to tame about our elements to achieve the desired results. We as developers need tools that help us adapt to this new form of animation to keep our HTML flexible.

Srcry provides 5 different size libraries and 2 configuration classes with custom properties for programming the scaling behavior of your elements. There’s no shortage of awesome UI libraries at our disposal so you won’t be seeing any of that here. The goal of this project is to provide an approachable means of creating complex scaling behaviors simply by defining a few properties which can be applied to any UI library that allows you to customize your own sizes. This demonstration doesn't do anything practical as it's just a demonstration, but the scaling behavior you just saw didn't require a single media query thanks to the srcryBox configuration class. All you need to do is add the srcryBox class to your HTML element and define the properties in your own css class and the srcryBox handles the rest.
